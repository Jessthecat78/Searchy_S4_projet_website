Searchy: a student-built search engine prototype

Searchy is our second-year project focused on information retrieval. The goal is to process user queries and return the most relevant results with a clear ranking strategy.

Quick overview

  • Project name: Searchy
  • Target: relevant results from user input
  • Current version: console-based prototype
  • Next step: complete web interface

Project intent

Relevance first

We focus on matching user intent and ranking outputs based on simple but efficient criteria.

Progressive build

The project evolves in steps: algorithm design, console tests, then interface integration and user testing.

Student approach

We intentionally keep the interface clean and straightforward to reflect an authentic second-year student project.
